              공통컴포넌트 개요

              공통컴포넌트 v4.3 가이드는 공통컴포넌트 v4.3에 대하여 실행환경 4.3 기반으로 변경된 사항을 적용한 버전으로 기능적인 차이보다는 내부 코드 품질 향상을 목적으로 작성되었으며 개요 부분과 컴포넌트별 상세 설명 부분으로 구성되었다.
