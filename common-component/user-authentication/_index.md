# 사용자인증

## 개요

사용자인증 카테고리는 아이디/비밀번호 기반 일반 로그인, 인증서(GPKI) 로그인, 로그인 정책 관리 기능을 제공한다. 소스는 주로 `egovframework.com.uat.*` 패키지에 위치한다.
