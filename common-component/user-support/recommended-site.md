## 개요

포털 또는 기관 웹사이트 방문자에게 유용한 외부 관련 기관, 유관 서비스, 협력사 등의 웹사이트 목록을 관리자가 추천 사이트로 등록하여 안내하는 컴포넌트입니다.

## 주요 기능

* **사이트 정보 등록**: 추천할 사이트의 명칭, URL 주소, 사이트에 대한 간략한 설명, 썸네일(로고) 이미지를 등록합니다.
* **추천 목록 제공**: 등록된 사이트들을 카테고리별 혹은 갤러리 형태로 사용자 화면에 리스팅하여 제공합니다.
* **승인/반려 관리**: (사용자 참여형일 경우) 사용자가 사이트를 추천하면 관리자가 이를 승인해야 목록에 노출됩니다.

## 데이터 구조 (테이블)

* `COMTNRECOMENDSITEINFO`: 추천 사이트 명칭, URL, 설명, 승인 여부, 등록 일자 등을 저장하는 테이블

## 활용 방안

공공기관의 '유관기관 안내' 페이지나 특정 주제(예: IT 기술 동향)를 다루는 포털의 '유용한 링크 모음' 페이지를 구축할 때 활용할 수 있습니다.
