## 개요

시스템의 로그인 페이지 배경이나 로그인 박스 주변 영역에 표시되는 홍보용 또는 디자인 이미지를 관리자가 직접 등록하고 교체할 수 있도록 지원하는 컴포넌트입니다.

## 주요 기능

* **이미지 등록**: 로그인 화면 전용 배경 이미지 또는 배너 이미지를 업로드합니다.
* **노출 순서 지정**: 다수의 이미지가 롤링(Slide)되는 구조일 경우 이미지 노출 순서를 설정합니다.
* **이미지 반영 제어**: 특정 시즌이나 행사에 맞춰 등록한 이미지를 사용 상태(Y/N) 값으로 제어하여 화면에 반영합니다.

## 데이터 구조 (테이블)

* `COMTNLOGINSCRENIAMGE`: 로그인 화면 이미지명, 파일 ID, 반영 여부 등을 저장하는 테이블

## 활용 방안

명절, 공휴일, 기관 창립일 등 특정 기념일이나 주요 행사가 있을 때, 개발자의 리소스 개입 없이 운영자가 로그인 화면의 디자인 리소스를 손쉽게 변경하여 시스템의 시각적 만족도를 높일 수 있습니다.
