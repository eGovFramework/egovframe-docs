## 개요

웹사이트 메인 화면이나 특정 페이지에 노출되는 배너 이미지와 연결 링크(URL)를 관리자가 손쉽게 등록, 수정, 노출 제어할 수 있도록 지원하는 컴포넌트입니다.

## 주요 기능

* **배너 이미지 업로드**: 배너로 사용할 이미지를 업로드하고 파일 정보를 저장합니다.
* **노출 순서 및 링크 설정**: 여러 배너가 있을 경우 정렬 순서를 지정하고, 클릭 시 이동할 타겟 URL을 설정합니다.
* **반영 여부 제어**: 특정 배너를 화면에 즉시 노출할지, 혹은 숨길지(Y/N) 상태 값을 통해 제어합니다.

## 데이터 구조 (테이블)

* `COMTNBANNER`: 배너 명칭, 링크 URL, 이미지 파일 ID, 노출 순서 등을 저장하는 테이블

## 활용 방안

포털 시스템의 롤링 배너, 공지사항 알림, 주요 행사 홍보 등 동적으로 화면의 배너 콘텐츠를 교체해야 할 때 소스 코드 수정 없이 관리자 페이지에서 바로 처리할 수 있습니다.
