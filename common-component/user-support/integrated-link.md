## 개요

자주 방문하는 유관 기관 웹사이트, 패밀리 사이트, 내부 시스템의 주요 메뉴 등에 대한 단축 링크(URL)를 카테고리별로 통합하여 관리하고 화면에 제공하는 컴포넌트입니다.

## 주요 기능

* **링크 정보 등록**: 사이트 명칭, 타겟 URL, 새 창 열기 여부 등의 링크 정보를 등록합니다.
* **카테고리 분류**: '관련기관', '패밀리사이트', '부서별 링크' 등 목적에 맞게 링크를 그룹화(카테고리)하여 관리합니다.
* **목록 제공**: 사용자가 접근할 수 있도록 포털 메인 화면의 셀렉트 박스나 푸터 영역 등에 링크 목록을 노출합니다.

## 데이터 구조 (테이블)

* `COMTNSYSTEMLINK`: 통합 링크 분류, 링크 URL, 링크 명칭 등을 저장하는 테이블

## 활용 방안

공공기관이나 기업 포털 하단의 '관련 사이트(Family Site)' 셀렉트 박스를 동적으로 구성하거나, 즐겨찾기 성격의 통합 링크 모음 페이지를 구축할 때 활용됩니다.
