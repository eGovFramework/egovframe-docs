## 개요

시스템의 각 화면이나 기능에 대한 사용 방법 및 안내 가이드를 온라인 도움말(Online Help) 형태로 사용자에게 제공하는 컴포넌트입니다.

## 주요 기능

* **도움말 콘텐츠 관리**: 각 메뉴 또는 화면별 기능 설명, 입력 방법 등에 대한 도움말 문서를 작성하여 등록합니다.
* **메뉴/화면 매핑**: 특정 메뉴를 클릭하거나 화면의 '?' (도움말 아이콘) 버튼을 클릭했을 때 관련된 도움말이 팝업이나 새 창으로 노출되도록 매핑합니다.

## 데이터 구조 (테이블)

* `COMTNONLINEHELP`: 도움말 구분, 도움말 제목, 본문 내용, 관련 메뉴 링크 등을 관리하는 테이블

## 활용 방안

사용자 매뉴얼 책자를 대체하여 시스템 내에서 실시간으로 사용자에게 기능 가이드를 제공함으로써, 시스템 학습 곡선을 낮추고 CS 문의를 감소시키는 데 활용됩니다.
