## 개요

단일 웹 애플리케이션 내에서 여러 개의 패밀리 사이트나 마이크로 사이트(Multi-tenant)를 운영할 경우, 각 사이트의 기본 메타 정보와 서비스 도메인을 분리하여 관리하는 컴포넌트입니다.

## 주요 기능

* **사이트 정보 등록**: 사이트 명칭, 접속 URL 도메인, 관리자 정보, 템플릿 테마 등을 등록하고 관리합니다.
* **운영 상태 제어**: 특정 하위 사이트의 서비스를 오픈하거나 임시 중단(점검 중 표시)하는 상태값을 제어합니다.
* **디자인 테마 매핑**: 각 사이트별로 적용할 공통 CSS, 로고 이미지, 푸터 정보 등을 개별적으로 매핑합니다.

## 데이터 구조 (테이블)

* `COMTNSITEMANAGE`: 사이트 ID, 도메인 정보, 사이트 명칭, 운영 상태 등을 관리하는 마스터 테이블

## 활용 방안

중앙 부처에서 산하 기관들의 홈페이지를 하나의 통합 CMS(콘텐츠 관리 시스템)로 관리하거나, 부서별로 성격이 다른 미니 사이트들을 생성하여 개별 도메인으로 서비스할 때 핵심적인 관리 기능으로 쓰입니다.
