## 개요

웹사이트 회원 가입이나 서비스 이용 시 사용자가 동의해야 하는 인터넷 서비스 이용 약관, 개인정보 취급 방침, 서비스 가이드 등을 등록하고 제공하는 컴포넌트입니다.

## 주요 기능

* **약관 내용 관리**: 서비스 이용약관, 개인정보 수집 및 이용 동의서 등의 본문 텍스트를 버전별로 등록/수정합니다.
* **사용자 노출**: 회원 가입 화면, 혹은 사이트 하단 푸터 영역에서 팝업 또는 새 페이지 형태로 사용자가 약관을 열람할 수 있게 제공합니다.

## 데이터 구조 (테이블)

* `COMTNSTPLATINFO`: 이용약관 구분 코드(이용약관, 개인정보보호정책 등), 약관 내용, 제정 일자 등을 저장하는 테이블

## 활용 방안

서비스 운영 상 법적으로 고지해야 할 필수 약관 정보들을 DB화하여 관리함으로써, 약관 개정 시 소스 코드 배포 없이 관리자 화면에서 즉각적으로 수정 내용을 반영할 수 있습니다.
