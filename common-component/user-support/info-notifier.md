## 개요

사용자에게 실시간 또는 주기적으로 공지사항, 시스템 점검, 이벤트 안내 등 주요 소식을 텍스트 롤링이나 팝업, 알림창 등의 형태로 제공하는 컴포넌트입니다.

## 주요 기능

* **알림 내용 등록**: 짧은 텍스트 형태의 알림 메시지와 클릭 시 이동할 URL 정보를 등록합니다.
* **알림 스케줄링**: 특정 기간(시작일시~종료일시)에만 알림이 노출되도록 기간을 설정합니다.
* **노출 방식 설정**: 팝업, 레이어, 화면 상단 띠 배너 등 알림을 노출하는 방식을 관리할 수 있습니다.

## 데이터 구조 (테이블)

* `COMTNINFOALRM`: 알림 메시지 내용, 적용 기간, 노출 여부 상태값 등을 저장하는 테이블

## 활용 방안

긴급 공지, 시스템 점검 시간 안내, 중요 업데이트 내역 등을 사용자가 시스템에 접속하자마자 즉각적으로 인지할 수 있도록 메인 화면 전면에 정보를 전달하는 기능으로 사용됩니다.
