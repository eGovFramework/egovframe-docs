## 개요

웹사이트의 통합검색 기능을 이용할 때, 개별 사용자가 최근에 입력했던 검색어 이력을 저장하고 다시 검색창을 클릭했을 때 목록으로 제공하는 컴포넌트입니다.

## 주요 기능

* **최근 검색어 저장**: 사용자가 검색을 수행할 때마다 해당 키워드를 쿠키(Cookie)나 DB에 저장합니다.
* **검색어 목록 제공**: 검색창에 포커스가 갈 때 사용자의 최근 검색어 목록을 레이어로 보여주어 빠른 재검색을 돕습니다.
* **검색어 삭제**: 사용자가 본인의 최근 검색어 이력을 개별 삭제하거나 전체 삭제할 수 있도록 지원합니다.

## 데이터 구조 (테이블)

(일반적으로 서버 부하를 줄이기 위해 브라우저 쿠키나 로컬 스토리지를 주로 활용하지만, 로그인 사용자의 기기 간 연동을 위해 별도 DB 테이블을 구성하기도 합니다.)

## 활용 방안

포털 사이트, 쇼핑몰, 커뮤니티 등의 통합검색 UI에서 검색 편의성을 높여주는 필수적인 사용자 인터페이스 요소로 사용됩니다.
