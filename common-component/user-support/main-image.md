## 개요

웹사이트 포털 메인(홈) 화면 상단의 주요 비주얼 영역(Key Visual)에 표시될 홍보 이미지 및 배너들을 등록하고 롤링되는 슬라이드 이미지를 관리하는 컴포넌트입니다.

## 주요 기능

* **메인 비주얼 이미지 등록**: 메인 화면 전용 고해상도 이미지를 업로드하고 설명 문구를 작성합니다.
* **순서 및 링크 설정**: 롤링되는 이미지의 노출 순서와 이미지를 클릭했을 때 이동할 상세 페이지 URL을 매핑합니다.
* **게시 기간 설정**: 이미지 노출 기간을 설정하여 이벤트 종료 시 자동으로 메인 화면에서 내려가도록 관리합니다.

## 데이터 구조 (테이블)

* `COMTNMAINIMAGE`: 메인 화면 이미지 파일 정보, 대체 텍스트(설명), 링크 정보 등을 관리하는 테이블

## 활용 방안

대국민 사이트 또는 그룹웨어 메인 화면에서 기관의 핵심 홍보 사항이나 최신 소식을 역동적으로 전달하는 메인 슬라이드(Carousel) 위젯을 제어하는 백오피스 기능으로 사용됩니다.
