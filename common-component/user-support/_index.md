# 사용자지원

## 개요

사용자지원 카테고리는 포털·홈페이지에서 일반 사용자에게 제공되는 서비스성 공통컴포넌트를 제공한다. 소스는 주로 `egovframework.com.uss.*` 패키지에 위치한다.

## 주요 영역

- **회원·개인화** — 일반/기업 회원 관리, 마이페이지, 개인화 설정, 사용자 부재 관리
- **참여·소통** — 설문조사, 온라인 참여(제안·모집), Q&A·상담, 헬프데스크(온라인매뉴얼·FAQ), 위키
- **콘텐츠·안내** — 공지·뉴스·배너·메인이미지, 사이트맵, 정보제공, 최근 검색어, RSS, 지도 안내
- **정책·기타** — 저작권보호·개인정보보호 정책, 약관 관리, 기념일·포상·당직·연차 관리, SNS(트위터·페이스북) 연계

각 기능의 상세 내용은 좌측 메뉴의 개별 문서를 참조한다.
