## 개요

기관의 보도자료, 언론 보도 내용, 내부 주요 소식 등을 뉴스 형식으로 웹사이트에 게시하고 관리할 수 있도록 지원하는 컴포넌트입니다.

## 주요 기능

* **뉴스 정보 등록**: 뉴스 제목, 요약 내용, 본문, 출처, 관련 이미지 및 첨부파일을 등록합니다.
* **게시물 관리**: 등록된 뉴스 콘텐츠의 노출 여부, 노출 순서 등을 제어합니다.
* **뉴스 목록/상세 제공**: 사용자 화면(Front)에 최신 뉴스 목록을 제공하고, 클릭 시 상세 본문이나 외부 언론사 링크로 이동하도록 지원합니다.

## 데이터 구조 (테이블)

* `COMTNNEWSINFO`: 뉴스 기사 제목, 내용, 출처 링크, 첨부파일 ID 등을 저장하는 테이블

## 활용 방안

홈페이지의 '보도자료', '뉴스룸(Newsroom)' 메뉴를 구성할 때 사용되며, 메인 화면의 뉴스 위젯(최근 게시물 노출) 기능과 연동하기 좋습니다.
