## 개요

웹사이트의 전체 메뉴 구조를 한 페이지 내에 트리(Tree) 또는 나열 형태로 보여주어, 사용자가 원하는 콘텐츠를 쉽고 빠르게 탐색할 수 있도록 돕는 컴포넌트입니다.

## 주요 기능

* **메뉴 구조 시각화**: DB에 등록된 메뉴 계층 정보(1 Depth, 2 Depth 등)를 읽어와 화면에 계층형 텍스트 구조로 출력합니다.
* **접근 권한 기반 필터링**: 일반 사용자, 로그인 사용자, 관리자 등 현재 접근한 사용자의 권한에 맞게 열람 가능한 메뉴만 사이트맵에 노출합니다.
* **XML 사이트맵 지원**: 검색 엔진 최적화(SEO)를 위해 크롤러 봇이 수집할 수 있는 sitemap.xml 형식의 데이터를 생성하기도 합니다.

## 데이터 구조 (테이블)

* `COMTNMENUINFO`: 시스템 메뉴 정보 테이블을 그대로 활용하여 전체 메뉴 트리를 구성합니다.

## 활용 방안

홈페이지 하단의 '사이트맵' 링크를 통해 사용자에게 내비게이션 뷰어 역할을 제공하며, 복잡한 메뉴 구조를 가진 포털 사이트에서 길잡이 역할을 수행합니다.
