## 개요

대한민국 행정안전부 등에서 제공하는 법정동(시/도, 시/군/구, 읍/면/동) 코드 체계를 시스템 내부에 수신하거나 등록하여 주소 관련 기능의 기초 데이터로 활용하는 컴포넌트입니다.

## 주요 기능

* **법정동코드 등록**: 파일 업로드나 연계를 통해 대량의 법정동코드를 일괄 등록하거나 수동으로 추가합니다.
* **코드 조회 및 검색**: 등록된 법정동코드를 지역명(예: '서초구', '역삼동' 등)으로 검색하고 계층 구조(시도 -> 시군구)를 확인합니다.
* **폐지 코드 관리**: 행정구역 개편으로 인해 통폐합되거나 폐지된 법정동코드를 이력으로 관리하여 과거 주소 데이터와의 호환성을 유지합니다.

## 데이터 구조 (테이블)

* `COMTNBASSCD`: 법정동코드, 지역명, 최하위 노드 여부, 폐지 여부 등을 저장하는 테이블

## 활용 방안

회원 가입 시 주소 검색, 부동산 관련 서비스, 지역 기반 통계 등 위치 정보가 필요한 시스템에서 표준화된 지역 코드를 제공하는 기반 모듈로 활용됩니다.
