# 메뉴관리

## 개요

메뉴관리는 프로그램목록에 등록된 프로그램 파일명을 기준으로 메뉴를 구성하고,
사용자가 접근할 수 있는 메뉴목록을 관리하는 기능이다. 최초 메뉴를 등록할 때는
프로그램 파일과 메뉴목록을 엑셀 파일로 정리하여 일괄 등록할 수 있다.

## 주요 개념

### 메뉴목록

메뉴목록은 화면에서 사용할 메뉴 정보를 계층 구조로 관리하는 정보이다. 등록된 메뉴는
메뉴명, 메뉴 순서, 상위 메뉴, 프로그램 파일명 등의 정보를 기반으로 구성된다.

### 프로그램 파일명

프로그램 파일명은 메뉴가 연결할 실제 화면 또는 기능을 식별하는 값이다. 메뉴 등록 또는
수정 시 프로그램 파일명을 검색하여 선택하고, 해당 파일명과 메뉴 정보를 연결한다.

### 메뉴일괄생성

메뉴일괄생성은 프로그램목록정보와 메뉴정보를 엑셀 파일로 작성하여 한 번에 등록하는
기능이다. 최초 메뉴목록을 구성할 때 편리하게 사용할 수 있으나, 기존 메뉴목록과
프로그램목록 데이터에 영향을 줄 수 있으므로 초기 등록 시점에 사용하는 것이 적합하다.

## 설명

메뉴관리 패키지는 요소기술의 공통 패키지와 프로그램관리 패키지를 직접 참조한다.
공통컴포넌트 배포 시에는 메뉴관리 기능이 정상적으로 동작하도록 관련 참조 패키지와
함께 구성해야 한다.

메뉴관리는 메뉴정보 목록조회, 메뉴정보 등록, 메뉴정보 상세조회 및 수정, 메뉴정보 삭제,
메뉴일괄생성, 메뉴리스트 관리 기능으로 구성된다. 관리자는 메뉴목록을 조회하고 필요한
메뉴를 등록하거나 수정할 수 있으며, 트리 형태의 메뉴리스트를 통해 상위 메뉴와 하위
메뉴의 관계를 관리할 수 있다.

## 관련 기능

- 메뉴정보 목록조회: 등록된 메뉴정보를 조회하고 검색한다.
- 메뉴정보 등록: 신규 메뉴정보를 등록하고 프로그램 파일명을 연결한다.
- 메뉴정보 상세조회/수정: 기존 메뉴정보를 확인하고 변경사항을 저장한다.
- 메뉴정보 삭제: 목록 또는 상세 화면에서 메뉴정보를 삭제한다.
- 메뉴일괄생성: 프로그램목록정보와 메뉴정보를 엑셀 파일로 일괄 등록한다.
- 메뉴리스트 관리: 메뉴를 트리 구조로 확인하고 상위 메뉴와 하위 메뉴 관계를 관리한다.
