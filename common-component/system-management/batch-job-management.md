## 개요

Spring Batch 등과 연계하여, 시스템 내에서 정기적으로 수행되어야 하는 대량 데이터 처리 작업(배치 프로그램)을 스케줄링하고 등록 관리하는 컴포넌트입니다.

## 주요 기능

* **배치 작업 등록**: 실행할 배치 프로그램(Job 이름), 실행 파라미터, 실행 주기(Cron 표현식) 등을 등록합니다.
* **작업 스케줄링 제어**: 등록된 배치의 스케줄러 등록을 활성화(Start)하거나 중지(Stop)하여 실행 상태를 제어합니다.
* **수동 실행**: 배치 스케줄러를 기다리지 않고 관리자가 화면에서 수동으로 즉시 배치를 실행할 수 있습니다.

## 데이터 구조 (테이블)

* `COMTNBATCHOPERT`: 배치 프로그램(작업)의 기본 정보 및 파라미터를 저장하는 테이블
* `COMTNSCHDULINFO`: 배치 작업의 실행 스케줄(Cron 주기 등) 정보를 저장하는 테이블

## 활용 방안

일일 통계 집계, 새벽 시간대 데이터 동기화, 만료 회원 일괄 탈퇴 등 사용자의 개입 없이 주기적으로 무거운 로직을 처리해야 할 때 핵심적인 스케줄 제어 화면으로 사용됩니다.
