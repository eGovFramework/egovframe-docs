## 개요

시스템 전체에서 사용될 메뉴 항목들의 마스터 정보를 등록하고, 각 메뉴가 어떤 프로그램(URL)을 호출할 것인지 매핑하는 기반 관리 컴포넌트입니다.

## 주요 기능

* **메뉴 마스터 등록**: 메뉴 번호, 메뉴명, 메뉴 설명, 상위 메뉴 번호 등을 등록하여 메뉴의 전체 풀(Pool)을 구성합니다.
* **프로그램 매핑**: 해당 메뉴를 클릭했을 때 실행될 프로그램 파일명(URL 또는 Controller Path)을 `프로그램관리`에서 선택하여 연결합니다.
* **메뉴 목록 관리**: 엑셀 업로드 방식 등을 통해 대량의 메뉴 구조를 일괄 등록하거나 갱신할 수 있습니다.

## 데이터 구조 (테이블)

* `COMTNMENUINFO`: 시스템 전체 메뉴 마스터 정보, 상위 메뉴 ID, 연결된 프로그램 파일명 등을 저장하는 테이블

## 활용 방안

개발자가 소스 코드에 메뉴를 하드코딩하지 않고, DB 기반으로 메뉴를 관리하게 함으로써 유지보수 시 화면 구조를 유연하게 변경할 수 있는 토대를 제공합니다.
