## 개요

시스템에서 사용되는 수많은 공통코드들을 대분류(업무 단위 등)로 묶어 관리하기 위한 최상위 계층의 '분류 코드(Class Code)'를 관리하는 컴포넌트입니다.

## 주요 기능

* **분류코드 등록/수정**: '회원관리', '게시판관리' 등 시스템의 큰 업무 도메인을 나타내는 분류 코드를 3자리 영문/숫자 등으로 등록합니다.
* **코드 계층 관리**: 분류코드 하위에 등록될 `공통코드(중분류)`들을 논리적으로 그룹화하는 기준점을 제공합니다.

## 데이터 구조 (테이블)

* `COMTNCCMMNCLCODE`: 공통분류코드 식별자, 분류코드명, 설명, 사용 여부를 저장하는 테이블

## 활용 방안

표준프레임워크의 코드 관리는 `분류코드(대) -> 공통코드(중) -> 상세코드(소)`의 3단계 계층 구조를 가집니다. 이 중 가장 상위에 해당하는 시스템 모듈별 기준 코드를 셋업할 때 사용됩니다.
