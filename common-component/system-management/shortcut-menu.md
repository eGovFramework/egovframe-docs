## 개요

시스템 사용자가 자주 이용하는 특정 메뉴(화면)나 게시판 등을 '즐겨찾기(Shortcut)'로 등록하여 메인 화면이나 퀵 메뉴 영역에서 빠르게 접근할 수 있도록 지원하는 컴포넌트입니다.

## 주요 기능

* **바로가기 등록/해제**: 사용자가 현재 보고 있는 화면을 마이페이지 또는 메인 화면의 바로가기 목록에 추가하거나 삭제할 수 있습니다.
* **순서 변경**: 등록된 바로가기 메뉴들의 노출 순서를 사용자가 원하는 대로 재배치합니다.
* **개인화 제공**: 사용자 계정별로 독립적인 바로가기 목록을 유지하여 개인화된 업무 환경을 제공합니다.

## 데이터 구조 (테이블)

* `COMTNBKMARK`: 사용자 ID별로 등록된 바로가기(북마크) 메뉴 ID와 정렬 순서를 저장하는 테이블

## 활용 방안

다양하고 복잡한 메뉴를 가진 인트라넷이나 ERP 시스템에서, 각 임직원이 본인의 담당 업무에 맞는 메뉴만을 추려내어 업무 효율성과 편의성을 높이는 데 활용됩니다.
