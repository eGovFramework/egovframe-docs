## 개요

국가 행정표준코드(법정동코드, 기관코드, 직급코드 등)를 시스템 내에서 체계적으로 수신하여 등록하고, 일반 공통코드와 구분하여 별도로 관리하기 위한 컴포넌트입니다.

## 주요 기능

* **행정표준코드 등록/수정**: 정부에서 제공하는 표준 행정 코드를 시스템에 등록하고, 변경 사항이 있을 시 수정합니다.
* **코드 목록 조회**: 등록된 행정코드의 목록을 조회하고, 코드명이나 코드 설명으로 필요한 코드를 검색합니다.
* **사용 여부 관리**: 행정 구역 개편 등으로 더 이상 사용되지 않는 코드는 삭제하지 않고 '사용 안함(N)' 처리하여 과거 데이터의 무결성을 유지합니다.

## 데이터 구조 (테이블)

* `COMTNADMINCODE`: 행정표준코드 아이디, 코드명, 사용 여부 등을 저장하는 테이블

## 활용 방안

대국민 서비스나 부처 간 연계 시스템에서 법정동 주소 검색, 행정 기관 선택 등 국가 표준 코드를 기준으로 작동해야 하는 기능의 마스터 데이터로 활용됩니다.
