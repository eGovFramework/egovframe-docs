## 개요

시스템 내부에서 사용될 기관, 본부, 팀 단위의 부서(조직) 마스터 코드 및 계층 구조를 등록하고 관리하는 컴포넌트입니다.

## 주요 기능

* **조직 마스터 등록**: 개별 부서의 고유 식별 코드, 부서명, 업무 역할 등을 수동으로 등록하고 수정합니다.
* **계층 구조 구성**: 상위 부서(예: 본부)와 하위 부서(예: 팀) 간의 관계를 매핑하여 사내 조직도 트리를 구성합니다.
* **조직 폐지 처리**: 더 이상 존재하지 않는 부서를 삭제하지 않고 '폐지' 상태로 변경하여 과거 문서 결재 이력 등을 보호합니다.

## 데이터 구조 (테이블)

* `COMTNINSTTCODE` / `COMTNORGNZTINFO`: 조직 코드 마스터 정보 및 상하위 트리 구조를 관리하는 테이블

## 활용 방안

내부 인트라넷, 그룹웨어, 전자결재 시스템 등에서 부서 기반의 접근 권한 제어(예: "우리 팀 게시물만 보기") 및 직원 소속 매핑을 위한 핵심 마스터 데이터로 활용됩니다.
