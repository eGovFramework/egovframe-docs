## 개요

시스템의 드롭다운(Select), 라디오버튼 등에 바인딩되어 실제 업무 데이터 값으로 사용되는 최하위 계층의 구체적인 '상세 코드(Detail Code)'를 관리하는 컴포넌트입니다.

## 주요 기능

* **상세코드 등록/수정**: 특정 공통코드 하위에 소속되는 실제 데이터 값(예: 성별 코드 하위의 'M: 남자', 'F: 여자')을 등록합니다.
* **정렬 순서 지정**: 화면에 렌더링될 때의 노출 순서를 코드별로 직접 지정하여 사용자 편의에 맞게 정렬할 수 있습니다.
* **사용 여부 관리**: 하위 코드가 더 이상 필요 없더라도 과거 데이터 조회를 위해 삭제 대신 '사용 안함'으로 처리합니다.

## 데이터 구조 (테이블)

* `COMTNCCMMNDETAILCODE`: 상위 공통코드 ID, 상세코드 ID, 상세코드명, 정렬 순서, 사용 여부 등을 저장하는 테이블

## 활용 방안

웹 애플리케이션의 화면 단에서 하드코딩을 제거하고, 동적으로 드롭다운 항목을 렌더링하거나 입력값을 검증(Validation)하는 데 있어 필수적인 데이터 딕셔너리로 광범위하게 쓰입니다.
