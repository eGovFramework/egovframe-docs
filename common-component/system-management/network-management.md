## 개요

시스템 운영 환경과 연관된 내부 또는 외부 서버 시스템들의 네트워크 접속 정보(IP, Port)를 마스터 데이터로 등록하고 관리하는 컴포넌트입니다.

## 주요 기능

* **시스템 정보 등록**: 데이터베이스 서버, 파일 서버, 외부 연계 기관 서버 등 시스템과 통신하는 대상 서버의 정보를 등록합니다.
* **네트워크 접속 관리**: 해당 시스템의 IP 주소, 통신 포트, 사용 프로토콜 등의 상세 네트워크 속성을 기록하고 변경 사항을 추적합니다.

## 데이터 구조 (테이블)

* `COMTNNTWRKINFO`: 네트워크 시스템 명칭, IP 주소, 포트, 용도 등을 저장하는 마스터 테이블

## 활용 방안

대규모 분산 환경이나 다양한 외부 기관과 API 연계를 수행하는 시스템에서, 타겟 서버의 연결 정보가 변경되었을 때 일괄적으로 참조하고 관리하기 위한 인프라 대장 역할을 수행합니다.
