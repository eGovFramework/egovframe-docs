## 개요

시스템 사용자가 로그인한 이후 특정 메뉴나 화면(URL)에 접근하여 조회, 등록 등의 서비스를 "사용한 내역"을 로깅하고 분석하는 데 목적을 둔 컴포넌트입니다.

## 주요 기능

* **서비스 사용 이력 수집**: 사용자가 어떤 기능(게시판 조회, 보고서 출력 등)을 수행했는지, 해당 서비스를 요청한 시간과 소요 시간을 기록합니다.
* **사용 통계 기초 데이터**: 기록된 사용 로그를 바탕으로 '가장 많이 사용된 메뉴', '시간대별 서비스 이용률' 등의 통계를 산출합니다.

## 데이터 구조 (테이블)

* `COMTNSUSELOG`: 서비스 호출 일시, 사용자 ID, 호출된 기능 URL, 응답 시간 등을 저장하는 테이블

## 활용 방안

단순한 접속 여부(로그인/로그아웃)를 넘어서, 사용자들이 시스템의 어떤 기능들을 주로 활용하고 있는지 행위 패턴을 분석하여 향후 시스템 고도화 및 메뉴 통폐합의 의사결정 자료로 활용됩니다.
