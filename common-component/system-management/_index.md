# 시스템관리

## 개요

시스템관리 카테고리는 시스템 운영자가 코드·메뉴·프로그램·로그·배치 등 시스템 자원을 관리하기 위한 공통컴포넌트를 제공한다. 소스는 주로 `egovframework.com.sym.*` 패키지에 위치한다.

## 주요 영역

- **코드 관리** — 공통분류코드·공통코드·공통상세코드, 행정코드(기관코드), 우편번호, 도로명주소
- **메뉴·프로그램 관리** — 메뉴 생성·관리, 바로가기 메뉴, 프로그램(화면) 관리
- **운영 관리** — 접속/시스템 로그 관리, 시스템 이력, 서버 정보, 네트워크 정보, 백업 관리
- **배치 관리** — 배치작업 등록, 배치 스케줄 처리, 배치 결과 관리

각 기능의 상세 내용은 좌측 메뉴의 개별 문서를 참조한다.
