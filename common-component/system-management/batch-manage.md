# 배치관리

## 개요

배치관리는 배치작업의 등록·관리, 스케줄에 따른 실행 처리, 실행 결과 관리를 제공하는 시스템관리 기능 묶음이다. 소스는 주로 `egovframework.com.sym.bat` 패키지에 위치한다.

## 관련 문서

- [배치작업관리](./batch-job-management.md) — 배치작업관리는 시스템에서 주기적으로 실행하는 배치작업을 등록하는 기능을 제공한다.
- [배치결과관리](./batch-result-management.md) — 배치결과관리는 시스템에서 주기적으로 실행하는 배치결과을 등록하는 기능을 제공한다.
- [스케줄처리](./schedule-processing.md)
