## 개요

시스템 내의 유용한 게시물(공지사항, 자료실 글 등)이나 외부 관심 정보를 사용자가 자신의 '개인 보관함(스크랩북)'에 복사/저장해 두고 나중에 다시 꺼내 볼 수 있도록 돕는 컴포넌트입니다.

## 주요 기능

* **게시물 스크랩**: 다른 사람이 작성한 유용한 게시글 본문 하단의 '스크랩' 버튼을 클릭하여 본인의 보관함으로 복사합니다.
* **스크랩 목록 관리**: 스크랩한 글들을 사용자가 직접 만든 폴더(분류)별로 정리하여 나만의 지식 창고를 구성합니다.
* **폴더 및 메모**: 스크랩 시 해당 글에 대한 나만의 코멘트(메모)를 덧붙이거나, '나중에 읽기', '중요 자료' 등의 폴더로 분류합니다.

## 데이터 구조 (테이블)

* `COMTNSCRAP`: 원본 게시물 ID, 스크랩한 사용자 ID, 개인 코멘트, 스크랩 등록 일시 등을 저장하는 테이블

## 활용 방안

포털 사이트나 대규모 커뮤니티에서 정보의 홍수 속에 사용자가 자신에게 필요한 정보만을 큐레이션(Curation)하여 업무나 개인 관심사에 활용할 수 있도록 지원하는 개인화 기능입니다.
