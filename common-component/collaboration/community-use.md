## 개요

시스템 관리자 관점에서 현재 운영 중인 여러 커뮤니티의 전체적인 운영 상태(사용 여부)를 제어하고, 각 커뮤니티에서 사용 중인 기능(게시판 등)을 매핑하는 컴포넌트입니다.

## 주요 기능

* **운영 상태 제어**: 특정 커뮤니티의 운영을 일시 정지하거나 폐쇄(사용 중지) 처리하여 사용자 접근을 차단합니다.
* **커뮤니티 메뉴/게시판 매핑**: 각 커뮤니티 공간 내에 노출될 게시판(예: 공지사항, 자유게시판, 자료실)을 할당하고 사용 여부를 관리합니다.
* **초기 스킨 및 템플릿 제어**: 시스템에서 제공하는 템플릿 중 해당 커뮤니티가 사용할 화면 레이아웃(스킨)을 지정합니다.

## 데이터 구조 (테이블)

* `COMTNCMMNTY`: 커뮤니티 마스터 테이블의 사용 여부(USE_AT) 속성을 업데이트합니다.
* `COMTNBBSUSE`: 해당 커뮤니티 ID와 게시판 마스터 ID 간의 사용(매핑) 관계를 저장하는 테이블

## 활용 방안

수많은 사내/외 커뮤니티가 생성되고 폐기되는 환경에서, 무분별하게 방치되는 커뮤니티 공간을 중앙 관리자가 효율적으로 모니터링하고 정리하는 통제 수단으로 활용됩니다.
