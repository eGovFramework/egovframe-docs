## 개요

단일한 테이블 구조를 바탕으로 공지사항, 자유게시판, 자료실, 갤러리(사진게시판) 등 다양한 성격의 게시판을 동적으로 생성하고 관리할 수 있도록 지원하는 프레임워크의 핵심 컴포넌트입니다.

## 주요 기능

* **다양한 UI 스킨 지원**: 일반 목록형, 답변형, 계층형, 앨범(갤러리)형 등 게시판의 목적에 맞는 화면 템플릿을 적용하여 렌더링합니다.
* **게시물 CRUD 및 첨부파일**: 웹 에디터를 통한 풍부한 본문 작성, 여러 개의 파일 업로드 및 다운로드, 게시물 수정/삭제 기능을 제공합니다.
* **권한 기반 제어**: 특정 권한을 가진 사용자만 글쓰기, 읽기, 답글 달기, 비밀글 작성이 가능하도록 게시판별로 접근 권한을 매핑합니다.

## 데이터 구조 (테이블)

* `COMTNBBS`: 실제 사용자가 작성한 게시물 본문, 조회수, 작성자 정보 등을 저장하는 통합 게시물 테이블
* `COMTNFILE`: 게시물에 첨부된 파일의 물리적/논리적 정보를 저장하는 파일 테이블과 매핑

## 활용 방안

새로운 게시판이 필요할 때마다 별도의 개발 없이 관리자 화면에서 '게시판 생성'만으로 즉시 새로운 소통 채널을 오픈할 수 있어, 개발 생산성을 극대화하는 표준 모듈입니다.
