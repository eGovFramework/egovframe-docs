# 문자메시지

## 개요

문자메시지는 전자정부 SMS 서비스(모바일 전자정부 M-Gov)와 연계하여 문자메시지를 전송하고 전송 내역을 관리하는 기능 묶음이다. 소스는 `egovframework.com.cop.sms` 패키지에 위치하며, 별도의 M-Gov 서비스 신청 후 사용할 수 있다.

## 관련 문서

- [문자메시지 서비스](./sms-service.md) — 전자정부 SMS 서비스(M-Gov)를 이용하기 위한 문자메시지 전송 인터페이스를 제공하며 별도의 M-Gov 신청을 통해 사용할 수 있다.
