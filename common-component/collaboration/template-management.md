## 개요

새로운 게시판이나 커뮤니티, 설문조사 등을 생성할 때 사용자 화면(UI)에 적용될 레이아웃 형태(디자인 스킨) 정보를 등록하고 관리하는 컴포넌트입니다.

## 주요 기능

* **템플릿 등록**: 템플릿의 고유 명칭(예: '게시판-갤러리형', '커뮤니티-기본형')과 실제 화면을 렌더링할 JSP 파일 물리 경로를 등록합니다.
* **템플릿 속성 관리**: 해당 템플릿이 어떤 서비스(게시판용, 커뮤니티용 등)에 적용될 수 있는지 구분 코드를 설정합니다.
* **사전 보기 기능**: 관리자가 템플릿을 선택할 때 디자인을 미리 가늠할 수 있도록 템플릿의 썸네일(미리보기 이미지)을 함께 등록하여 제공할 수 있습니다.

## 데이터 구조 (테이블)

* `COMTNTMPLATINFO`: 템플릿 ID, 템플릿 명칭, 템플릿 파일 경로, 템플릿 구분(게시판/커뮤니티) 등을 저장하는 테이블

## 활용 방안

`게시판속성관리`나 `커뮤니티생성` 시 관리자가 화면 렌더링 방식을 유연하게 선택할 수 있도록 목록을 제공하며, 디자인 변경 시 소스 수정 없이 템플릿 매핑만 변경하여 UI를 쉽게 교체하는 데 기여합니다.
