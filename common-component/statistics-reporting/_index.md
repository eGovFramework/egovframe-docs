# 통계리포팅

## 개요

통계리포팅 카테고리는 시스템 이용 현황을 집계하여 기간별(연도별·월별·일별) 그래프와 텍스트로 조회하는 통계 공통컴포넌트를 제공한다. 소스는 주로 `egovframework.com.sts.*` 패키지에 위치한다.
