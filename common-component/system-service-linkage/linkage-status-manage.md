## 개요

외부 시스템과의 데이터 연계(API 호출, 전문 송수신) 작업이 정상적으로 수행되고 있는지 종합적으로 모니터링하고, 발생한 트랜잭션의 송수신 상태와 통계 정보를 실시간으로 조회하는 컴포넌트입니다.

## 주요 기능

* **송수신 상태 모니터링**: 개별 연계 작업의 처리 상태(대기, 송신 중, 정상 완료, 오류 발생)를 조회하여 인터페이스의 정상 동작 여부를 확인합니다.
* **오류 메시지 추적**: 연계 실패 시 상대 기관 시스템에서 반환한 에러 코드(예: 타임아웃, 데이터 형식 오류)와 실패한 원본 메시지를 확인하여 장애 조치를 지원합니다.
* **연계 통계 제공**: 일별, 월별, 기관별 연계 발생 건수와 성공률 통계를 차트나 리스트 형태로 제공합니다.

## 데이터 구조 (테이블)

* `COMTNSYSTEMCNTC`: 연계 ID별 실시간 통신 상태 및 누적 송수신 건수를 관리하는 현황 테이블 (송수신 이력 로그 테이블과 조인하여 활용)

## 활용 방안

시스템 관제실이나 IT 운영 부서에서 대외 연계망의 상태를 실시간으로 파악하는 대시보드의 기초 데이터로 활용되며, 잦은 타임아웃이 발생하는 연계 구간을 식별하여 시스템 성능을 튜닝하는 데 쓰입니다.
