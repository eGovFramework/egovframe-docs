# 시스템/서비스연계

## 개요

시스템/서비스연계 카테고리는 기관 간 시스템 연계를 위한 연계기관·연계시스템 등록, 연계현황 관리, 연계메시지 관리 기능을 제공한다. 소스는 주로 `egovframework.com.ssi.*` 패키지에 위치한다.
