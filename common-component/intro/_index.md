# 공통컴포넌트 개요

## 개요

전자정부 표준프레임워크 공통컴포넌트는 전자정부 시스템에서 공통적으로 사용되는 기능을 재사용 가능한 형태로 제공하는 컴포넌트 모음이다. 이 절은 공통컴포넌트를 도입·활용하기 위한 공통 기반 정보를 제공한다.
