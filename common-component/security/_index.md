# 보안

## 개요

보안 카테고리는 시스템 자원에 대한 접근을 통제하기 위한 권한 관리 공통컴포넌트를 제공한다. 권한과 롤(Role)을 정의하고 사용자·그룹에 부여하여 기능 접근을 제어한다. 소스는 주로 `egovframework.com.sec.*` 패키지에 위치한다.
