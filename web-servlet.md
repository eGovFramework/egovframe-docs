# Web Servlet

### 1. 웹 서블릿의 기본 개념과 목적 : 
#### 웹 애플리케이션에서 클라이언트 요청을 처리하고 동적 콘텐츠를 생성하기 위한 자바 클래스

### 2. 서블릿의 생명주기 : 
#### 초기화(init), 서비스(service), 소멸(destroy) 단계로 구성됨



### 3. 요청 처리 및 응답 생성 방법 : 
#### HttpServletRequest 객체로 요청을 받고, HttpServletResponse 객체로 응답을 생성

### 6. 서블릿과 관련된 주요 클래스와 인터페이스 :
