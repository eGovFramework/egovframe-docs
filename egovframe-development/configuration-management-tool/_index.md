# 형상 관리 도구

## 개요

형상 관리 도구는 소스코드의 버전 관리와 변경·이슈 관리를 지원한다. 소스관리도구의 클라이언트·서버 환경 운영 방법과 Git 기반 운영, 이슈관리 도구의 활용 방법을 제공한다.

## 주요 개념

형상관리(Configuration Management)는 소스코드의 변경 이력을 추적·관리하는 활동이며, 변경관리(Change Management)·이슈관리와 연계해 운영한다. 소스관리도구는 클라이언트 환경(개발자)과 서버 환경(운영자)으로 나누어 구성한다.
