# 개발환경 설치 가이드

## 개요

개발환경 설치 가이드는 개발자 개발환경과 서버 개발환경을 구성하기 위한 설치 절차를 안내한다. 구현 도구의 수동 설치와 VS Code Extension 설치 방법도 함께 제공한다.

## 주요 개념

개발자는 구현 도구(Eclipse 기반) 또는 VS Code Extension 중 환경에 맞는 도구를 설치해 사용하며, 서버 개발환경은 형상관리·통합빌드 등 공동 작업에 필요한 서버를 구성한다.
