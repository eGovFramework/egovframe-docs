Mock к°қмІҙмқҳ мқҳлҜё

Mock к°қмІҙ( Mock Object - wikipedia , mock - л„ӨмқҙлІ„ мҳҒм–ҙмӮ¬м „ )лҠ” Unit Testмқҳ лҸ…лҰҪм„ұ(isolation)мқ„ лҶ’м—¬мЈјкё° мң„н•ҙ мӮ¬мҡ©н•ҳл©°, н…ҢмҠӨнҠён•ҳкі мһҗ мҪ”л“ңмҷҖ кҙҖл Ёмқҙ мһҲлҠ” к°қмІҙ(collaborator)лҘј нқүлӮҙлӮҙм–ҙ Unit TestлҘј мҲҳн–үн•  мҲҳ мһҲлҸ„лЎқ лҸ„мҷҖмЈјлҠ” к°қмІҙмқҙлӢӨ.

лӢӨмқҢкіј к°ҷмқҙ Programм—җм„ң ObjectлқјлҠ” к°қмІҙлҘј н…ҢмҠӨнҠён•ҳкі мһҗ н•  кІҪмҡ°, к·ё ObjectлҠ” лӢӨлҘё нҒҙлһҳмҠӨмҷҖ кҙҖкі„лҘј к°–кі  мһҲкІҢ л§Ҳл ЁмқҙлӢӨ.
к°ңл°ңмһҗлҠ” Object н•ҳлӮҳл§Ңмқ„ н…ҢмҠӨнҠён•ҳкі  мӢ¶кі , м—¬лҹ¬ к°Җм§Җ мқҙмң лЎң Collaborator1, Collaborator2, Collaborator3 к°қмІҙлҘј мӮ¬мҡ©н•  мҲҳк°Җ м—ҶлӢӨкі  н•ңлӢӨл©ҙ,
м»ҙнҢҢмқјм—җ м§ҖмһҘмқҙ м—ҶлҸ„лЎқ CollaboratorлҘј нқүлӮҙлӮҙлҠ” Mock к°қмІҙ (MockCollaborator1, MockCollaborator2, MockCollaborator3)лҘј мғқм„ұн•ҳкі  Objectк°Җ мӮ¬мҡ©н•ҳлҠ” л©”мҶҢл“ңм—җ лҢҖн•ҙм„ңл§Ң мӣҗн•ҳлҠ” лҸҷмһ‘мқ„ м •мқҳн•ң л’Ө, ObjectлҘј н…ҢмҠӨнҠён•ҳлҠ” кІғмқҙлӢӨ.

мўҖлҚ” мү¬мҡҙ мҳҲлҘј л“Өмһҗл©ҙ, н…ҢмҠӨнҠё лҢҖмғҒ нҒҙлһҳмҠӨмқҳ м»ҙнҢҢмқј мҳӨлҘҳлҘј м—Ҷм• кё° мң„н•ҙм„ң мһ„мӢңлЎң Mock к°қмІҙлҘј мғқм„ұн•  мҲҳлҸ„ мһҲлӢӨ. нҳ№мқҖ мӣ№м»Ён…Ңмқҙл„ҲлӮҳ DB нҷҳкІҪ л“ұмқ„ мӮ¬мҡ©н•ҙм•ј н•ҳлҠ”лҚ°, к·ё нҷҳкІҪмқ„ кө¬м„ұн•ҳкё°к°Җ м–ҙл өлӢӨл©ҙ, н•ҙлӢ№ нҷҳкІҪмқ„ нқүлӮҙлӮҙлҠ” нҷҳкІҪмқ„ Mockingн•ҳкұ°лӮҳ мқҙлҜё Mockingн•ң нҷҳкІҪмқ„ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ. мқҙлҹ¬н•ң лӘЁл“  кІғл“Өмқ„ Mockмқҙлқј н•ңлӢӨ.

  • н…ҢмҠӨнҠёк°Җ м§„м§ң к°қмІҙм—җкІҢ к·ёкІғмқҙ м–ҙл–»кІҢ мӮ¬мҡ©лҗҳм—ҲлҠ”м§Җ л¬јм–ҙліҙм•„м•ј н•ңлӢӨ.

  1. мқҙлҜё кө¬нҳ„лҗң Mock к°қмІҙ мӮ¬мҡ©
    JEE нҷҳкІҪмқҳ мӣ№м»Ён…Ңмқҙл„ҲлӮҳ DB, л©”мқјм„ңлІ„ л“ұмқҖ м•„мЈј нқ”нһҲ мӮ¬мҡ©лҗҳлҠ” н…ҢмҠӨнҠён•ҳкё° м–ҙл Өмҡҙ нҷҳкІҪмқҙлӢӨ. мқҙлҹ¬н•ң кІғл“ӨмқҖ мқҙлҜё мҳӨн”ҲмҶҢмҠӨлӮҳ мғҒмҡ© S/WлЎң Mock нҷҳкІҪмқҙлӮҳ Mock к°қмІҙлҘј м–ҙлҠҗ м •лҸ„ кө¬нҳ„н•ҳм—¬ м ңкіөн•ҳкё°лҸ„ н•ңлӢӨ. нҠ№нһҲ мӢӨн–үнҷҳкІҪм—җм„ң мұ„нғқн•ң Springм—җм„ңлҠ” н…ҢмҠӨнҠё нҒҙлһҳмҠӨлҘј мһ‘м„ұн•ҳкё° мүҪлҸ„лЎқ лӢӨм–‘н•ң Mock к°қмІҙмҷҖ мң нӢёлҰ¬нӢ°лҘј м ңкіөн•ңлӢӨ.

    вң” мӮ¬мҡ©лІ•кіј мғҳн”Ңм—җ лҢҖн•ң м„ӨлӘ…мқҖ мқҙлҜё кө¬нҳ„лҗң Mock к°қмІҙ мӮ¬мҡ©мқ„ м°ёмЎ°н•ңлӢӨ.
  2. Mock лқјмқҙлёҢлҹ¬лҰ¬ мӮ¬мҡ©
    к°„лӢЁн•ң нҒҙлһҳмҠӨлӮҳ мқён„°нҺҳмқҙмҠӨк°Җ мһҲлҠ” кІҪмҡ°, м»ҙнҢҢмқј мҳӨлҘҳлҘј л°©м§Җн•ҳкё° мң„н•ҙм„ңлӮҳ нғҖ к°қмІҙм—җ лҢҖн•ң мқҳмЎҙм„ұмқ„ мӨ„мқҙкё° мң„н•ҙ мһҗмӢ мқҳ TestCase мҪ”л“ң лӮҙм—җм„ң мһ мӢң нғҖ к°қмІҙм—җ лҢҖн•ҙ Mockingн•  мҲҳ мһҲлҸ„лЎқ м ңкіөн•ҳлҠ” лӢӨм–‘н•ң мҳӨн”ҲмҶҢмҠӨ к°Җ мһҲлӢӨ. ліё к°ңл°ңнҷҳкІҪм—җм„ңлҠ” EasyMock мқ„ мӮ¬мҡ©н•ңлӢӨ.
    мҳҲ) http://www.mockobjects.com/ , EasyMock , JMock , Mockito л“ұ
    вң” мӮ¬мҡ©лІ•кіј мғҳн”Ңм—җ лҢҖн•ң м„ӨлӘ…мқҖ Mock лқјмқҙлёҢлҹ¬лҰ¬ мӮ¬мҡ©мқ„ м°ёмЎ°н•ңлӢӨ.
  3. MockмңјлЎң мӮ¬мҡ©н•  CollaboratorлҘј м§Ғм ‘ кө¬нҳ„
    мҪ”л“ң лӮҙм—җм„ң мһҗмЈј Mockingлҗҳкұ°лӮҳ м—¬лҹ¬ кіім—җм„ң мӮ¬мҡ©лҗҳлҠ” нҒҙлһҳмҠӨлӮҳ н…ҢмҠӨнҠёлҘј мң„н•ң нҷҳкІҪмқ„ лі„лҸ„лЎң мһ‘м„ұн•ҙлҶ“кі  мӢ¶мқҖ кІҪмҡ°м—җ м§Ғм ‘ Mock нҒҙлһҳмҠӨлҘј мғқм„ұн•  мҲҳ мһҲлӢӨ.

    вң” мӮ¬мҡ©лІ•кіј мғҳн”Ңм—җ лҢҖн•ң м„ӨлӘ…мқҖ MockмңјлЎң мӮ¬мҡ©н•  CollaboratorлҘј м§Ғм ‘ кө¬нҳ„ мқ„ м°ёмЎ°н•ңлӢӨ.

н…ҢмҠӨнҠё лҢҖмғҒ мҪ”л“ңк°Җ мӮ¬мҡ©н•ҳлҠ” к°„лӢЁн•ң нҒҙлһҳмҠӨлӮҳ мқён„°нҺҳмқҙмҠӨм—җ лҢҖн•ҙ Mock нҒҙлһҳмҠӨлҘј мһ‘м„ұн•ҳкі  мһҲкё°ліҙлӢӨлҠ” кҙҖмӢ¬мһҲлҠ” л©”мҶҢл“ңм—җ лҢҖн•ҙм„ңл§Ң Mockingн•ҳм—¬ лӮҙк°Җ н…ҢмҠӨнҠён•ҳлҠ” мқјм—җ м§ҖмһҘмқҙ м—ҶлҸ„лЎқ н•  мҲҳ мһҲлҠ” Mock Frameworkмқҙ лӘҮ к°Җм§Җ мһҲлӢӨ. к·ё мӨ‘ EasyMockмқ„ мӮ¬мҡ©н•  мҲҳ мһҲлҠ”лҚ°, ліё к°ңл°ңнҷҳкІҪм—җм„ңлҠ” EasyMockкіј н•Ёк»ҳ UnitilsлҘј н•Ёк»ҳ мӮ¬мҡ©н•  кІҪмҡ° лҚ”мҡұ к°„лӢЁн•ҳкІҢ Mockingн•  мҲҳ мһҲлӢӨ. EasyMock мӮ¬мҡ©м—җ лҢҖн•ҙм„ңлҠ” EasyMock Documentation мқҙлӮҳ к°Ғмў… кҙҖл Ё лё”лЎңк·ёлҘј нҷ•мқён•ҳкё° л°”лһҖлӢӨ.

EasyMock мӮ¬мҡ© мҳҲ

м—¬кё°м—җм„ңлҠ” EasyMockкіј UnitilsлҘј н•Ёк»ҳ мӮ¬мҡ©н•ң мҪ”л“ңм—җ лҢҖн•ҙ м„ӨлӘ…н•ҳлҸ„лЎқ н•ҳкІ лӢӨ. EasyMockTest мҶҢмҠӨ м—җм„ң м „мІҙ мҶҢмҠӨлҘј нҷ•мқён•ҳкі  л№„көҗн•ҙліҙкё° л°”лһҖлӢӨ. Unitilsмқҳ Mock м§Җмӣҗ кҙҖл Ён•ҙм„ңлҠ” Unitils Tutorial - Testing with Mock objects мҷҖ EasyMock Support лҘј м°ёмЎ°н•ҳкё° л°”лһҖлӢӨ.

к°ҷмқҖ кІҪмҡ°м—җ лҢҖн•ҙ EasyMockл§Ңмқ„ мӮ¬мҡ©н•ҳм—¬ к°ңл°ңн•ң кІҪмҡ°м—җ лҢҖн•ҙм„ңлҠ” EasyMockл§Ң мӮ¬мҡ©н•ң кІҪмҡ° лҘј м°ёмЎ°н•ҳм—¬ л№„көҗн•ҙліҙкё° л°”лһҖлӢӨ.

ліё мғҳн”Ңм—җм„ңлҠ” CollaboratorDaoлҘј мӮ¬мҡ©н•ҳкі  мһҲлҠ” EasyMockServiceм—җ лҢҖн•ң н…ҢмҠӨнҠё мҪ”л“ң мҳҲлҘј лӢҙм•ҳлӢӨ. MockмңјлЎң л§Ңл“ңлҠ” лҢҖмғҒмқҖ мқён„°нҺҳмқҙмҠӨл§Ң мһҲмңјл©ҙ м¶©л¶„н•ҳлҜҖлЎң CollaboratorDaoмқҖ мқён„°нҺҳмқҙмҠӨмқҙкі  к·ё кө¬нҳ„ нҒҙлһҳмҠӨлҠ” м–ҙл–Ө кІғмқҙ мҷҖлҸ„ мғҒкҙҖмқҙ м—ҶлӢӨ.

н…ҢмҠӨнҠё лҢҖмғҒ мҪ”л“ң(EasyMockService)лҠ” лӢӨмқҢкіј к°ҷлӢӨ. selectListмҷҖ insert л©”мҶҢл“ңлҘј к°–кі  мһҲкі , к°Ғк°Ғ CollaboratorDaoмқҳ selectListмҷҖ insert л©”мҶҢл“ңлҘј нҳём¶ңн•ҳкі  мһҲлӢӨ.

@Service("easyMockService")
public class EasyMockService {

    @Resource(name="collaboratorDao")
    private CollaboratorDao collaboratorDao;

    public List<BoardVO> selectList() throws Exception {
        List<BoardVO> result =  collaboratorDao.selectList();
        return result;
    }

    public void insert(BoardVO vo) throws Exception {
        collaboratorDao.insert(vo);
    }

}

CollaboratorDaoлҠ” лӢӨмқҢкіј к°ҷмқҖ мқён„°нҺҳмқҙмҠӨмқҙлӢӨ.

public interface CollaboratorDao {
    List<BoardVO> selectList();
    void insert(BoardVO vo);
}

н…ҢмҠӨнҠё мҪ”л“ңлҠ” лӢӨмқҢкіј к°ҷмқҙ мһ‘м„ұн•  мҲҳ мһҲлӢӨ.

  1. Unitils кё°л°ҳ н•ҳм—җ н…ҢмҠӨнҠёлҘј мҲҳн–үн•  кІғмһ„мқ„ м„ м–ён•ңлӢӨ.

    @RunWith(UnitilsJUnit4TestClassRunner.class)  
public class EasyMockTest {  
    . . .  
}

  2. Mock лҢҖмғҒмқ„ м„ м–ён•ңлӢӨ.

    • @Mock : MockмңјлЎң м •мқҳн•Ё
    • @InjectIntoByType : CollaboratorDaoк°Җ Service нҒҙлһҳмҠӨм—җ InjectлҗЁ.
    @Mock  
@InjectIntoByType  
private CollaboratorDao mockDao;

  3. н…ҢмҠӨнҠё лҢҖмғҒ нҒҙлһҳмҠӨлҘј м„ м–ён•ңлӢӨ.

    • @TestedObject : н…ҢмҠӨнҠё лҢҖмғҒ нҒҙлһҳмҠӨлЎң Mock к°қмІҙк°Җ Injectлҗ  лҢҖмғҒмқҙлӢӨ.
    @TestedObject

  4. Mockingн•ҳкі мһҗ н•ҳлҠ” л©”мҶҢл“ңлҘј м •мқҳн•ңлӢӨ.
    мқҙлҠ” н…ҢмҠӨнҠё лҢҖмғҒ л©”мҶҢл“ңмҷҖ кҙҖл Ёлҗң л©”мҶҢл“ңм—җ лҢҖн•ҙ лҜёлҰ¬ м •мқҳлҘј н•ҙл‘җлҠ” кІғмқҙлӢӨ. н•ҙлӢ№ л©”мҶҢл“ңмқҳ signatureлҘј м •мқҳн•ҳкі  лҰ¬н„ҙк°’мқҙ мһҲлҠ” кІҪмҡ° лҰ¬н„ҙк°’мқ„ м •н•ҙлҶ“кі  к·ё лҚ°мқҙн„°лҘј мқҙмҡ©н•ҳм—¬ н…ҢмҠӨнҠёлҘј м§„н–үн•ҳлҠ” кІғмқҙлӢӨ.

    • Mocking лҢҖмғҒ л©”мҶҢл“ңк°Җ лҰ¬н„ҙк°’мқ„ к°–м§Җ м•ҠлҠ” кІҪмҡ°лҠ” лӢӨмқҢкіј к°ҷмқҙ н•ңлӢӨ.

      mockDao.insert(board);  
EasyMockUnitils.replay();

    • Mocking лҢҖмғҒ л©”мҶҢл“ңк°Җ лҰ¬н„ҙк°’мқ„ к°–лҠ” кІҪмҡ°лҠ” лӢӨмқҢкіј к°ҷмқҙ н•ңлӢӨ.

      expect(mockDao.selectList()).andReturn(Arrays.asList(new BoardVO(101), new BoardVO(102)));  
EasyMockUnitils.replay();

  5. н…ҢмҠӨнҠё лҢҖмғҒ л©”мҶҢл“ңлҘј мӢӨн–үн•ҳкі  мқҙнӣ„ н…ҢмҠӨнҠёлҘј нҷ•мқён•ҳлҠ” л“ұмқҳ мҪ”л“ңлҘј мһ‘м„ұн•ңлӢӨ.

// кІ°кіјлҘј нҷ•мқён•ңлӢӨ.

assertPropertyLenientEquals("id", Arrays.asList(101, 102), selectList);

MockмңјлЎң мӮ¬мҡ©н•  CollaboratorлҘј м§Ғм ‘ кө¬нҳ„

к°Җмқҙл“ң н”„лЎңк·ёлһЁм—җлҠ” н”„лЎңм қнҠём—җм„ң н•„мҡ”лЎң н•ҳлҠ” кіөнҶөлҗң Mock нҒҙлһҳмҠӨлҘј мһ‘м„ұн•ҳлҠ” кІҪмҡ°м—җ лҢҖн•ҙ мғҳн”Ң мҪ”л“ңлҘј лӢҙм•ҳлӢӨ. EmailMockTestмҷҖ JDBCMockTestк°Җ к·ёкІғмқёлҚ°, к·ё лӮҙмҡ©мқ„ мқјмқјмқҙ м„ӨлӘ…н•ҳкё°ліҙлӢӨлҠ” Mockмқ„ м§Ғм ‘ мһ‘м„ұн•ҳлҠ” кІҪмҡ°мқҳ мқјл°ҳм Ғмқё л°©лІ•м—җ лҢҖн•ҙм„ңл§Ң к°„лӢЁнһҲ м–ёкёүн•ҳлҸ„лЎқ н•ҳкІ лӢӨ. к·ё мҷём—җлҠ” мқјл°ҳм Ғмқё мҪ”л“ңлҘј мһ‘м„ұн•ҳлҠ” кІғкіј лӢӨлҘҙм§Җ м•ҠлӢӨ.

  1. Mockingн•ҳкі мһҗ н•ҳлҠ” мқён„°нҺҳмқҙмҠӨлҘј мғҒмҶҚл°ӣлҠ” нҒҙлһҳмҠӨлҘј мғқм„ұн•ңлӢӨ.
  2. мқён„°нҺҳмқҙмҠӨм—җ м •мқҳлҗң л©”мҶҢл“ңлҘј кө¬нҳ„н•ҳлҗҳ н…ҢмҠӨнҠё лҢҖмғҒм—җм„ң мӮ¬мҡ©н•ҳлҠ” л©”мҶҢл“ңл§Ң кө¬нҳ„н•ңлӢӨ.
    мқҙ л•Ң, кө¬нҳ„ лӮҙмҡ©мқҖ мӣҗн•ҳлҠ” лҚ°мқҙн„°лҘј лӢЁмҲңнһҲ лҰ¬н„ҙн•ҳкұ°лӮҳ мқјл¶Җ н•„мҡ”н•ң лЎңм§Ғл§Ң кө¬нҳ„н•ҳлҠ” м •лҸ„мқҙл©ҙ лҗңлӢӨ.
