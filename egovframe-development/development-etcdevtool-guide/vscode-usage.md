# VS Code 활용 가이드

## 서론

본 가이드는 Visual Studio Code(이하 VS Code)에서 확장 설치 방법과 환경 설정 방법, 프로젝트 구동 방법을 안내한다.

## 요구사항

- Visual Studio Code 1.84.0 이상, 최신 버전 권장
- JDK 17 (VS Code 버전에 따라서 JDK 21 이상도 추가로 필요)
- Maven 3.9.9 (권장)
- Tomcat 10.0.x, Tomcat 10.1.x
