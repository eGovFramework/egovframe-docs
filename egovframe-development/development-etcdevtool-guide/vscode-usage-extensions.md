# VS Code 확장 설치 가이드

## 개요

VS Code에서 전자정부 표준프레임워크 프로젝트를 원활히 활용하기 위해서는 여러 확장(Extension)을 설치해야 한다.

확장 설치 방법은 다음 2가지 중에서 선택할 수 있다.
1. VS Code Marketplace에서 개별 설치
2. egovframe 프로필을 가져와 일괄 설치

### 설치 방법 1. VS Code Marketplace에서 개별 설치

확장 개별 설치 방법은 [VS Code Extension 설치 가이드 - VS Code Marketplace에서 설치](../install-guide/vscode-extension-install-guide.md#설치-방법-1-vs-code-marketplace에서-설치)을 참조한다.

위 방식으로 반드시 설치해야 하는 확장 목록은 아래와 같다.

#### eGovFrame VSCode Extension
- eGovFrame Initializr in VSCode / eGovFramework

#### Java Extensions
- Extension Pack for Java / Microsoft
- Language Support for Java ™ for Visual Studio Code / Red Hat
- Project Manager for Java / Microsoft
- Maven for Java / Microsoft
- Gradle for Java / Microsoft
- Test Runner for Java / Microsoft
- Debugger for Java / Microsoft

#### Spring & Spring Boot Extensions
- Spring Boot Dashboard / Microsoft
- Spring Boot Tools / VMware
- Spring Initializr Java Support / Microsoft

#### Server Extensions
- Community Server Connectors / Red Hat
- Runtime Server Protocol UI / Red Hat

#### 기타
- Properties Editor / kuju63

### 설치 방법 2. egovframe 프로필을 가져와 일괄 설치

VS Code용 egovframe 프로필을 이용해 확장들을 일괄 설치하는 방법은 [VS Code Extension 설치 가이드 - egovframe 프로필](../install-guide/vscode-extension-install-guide.md#설치-방법-3-egovframe-프로필을-가져와-일괄-설치)을 참조한다.

앞서 열거한 필수 확장 외에도 편의 기능을 제공하는 추가 확장들이 함께 설치된다.
