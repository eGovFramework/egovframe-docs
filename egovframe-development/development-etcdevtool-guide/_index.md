# 개발환경 활용 가이드

## 개요

개발환경 활용 가이드는 표준프레임워크 구현 도구 외의 개발 도구를 활용하는 방법을 안내한다. IntelliJ와 VS Code 활용 가이드를 제공한다.

## 주요 개념

표준프레임워크 프로젝트는 Eclipse 기반 구현 도구 외에도 IntelliJ, VS Code 등 개발자가 선호하는 도구에서 개발할 수 있으며, 각 도구별 환경 설정·확장 설치·프로젝트 구동 방법을 안내한다.
