# 개발환경 소개

## 개요

표준프레임워크 개발환경은 응용SW를 개발·구현·시험·배포하는 데 필요한 도구와 환경을 제공한다. 개발자 PC에 구성하는 개발자 개발환경과, 형상관리·통합빌드 등을 담당하는 서버 개발환경으로 구분된다.

## 주요 개념

개발환경은 구현 도구(IDE), 테스트 도구, 형상 관리 도구, 배포 도구로 구성되며, 개발자는 개발자 개발환경을, 운영자는 서버 개발환경을 구성해 사용한다.
