# 실행환경 소개

## 개요

표준프레임워크 실행환경은 응용SW의 구성 기반으로서, 응용SW 실행 시 필요한 기본 기능을 제공하는 환경을 의미한다. 화면처리, 업무처리, 데이터처리, 연계통합, 배치처리, 공통기반으로 구성된 레이어드 아키텍처를 제공한다.

## 주요 개념

실행환경은 계층별로 역할이 분리되어 있어 각 계층을 독립적으로 이해하고 적용할 수 있다. 최근에는 클라우드 전환·컨테이너 배포·프론트엔드 분리·MSA·이벤트 기반 연계 등을 함께 고려하는 경우가 늘고 있어, 기존 레이어드 구조를 유지할 경우와 클라우드 환경에 맞는 구조를 검토할 경우를 구분해 이해할 필요가 있다.
