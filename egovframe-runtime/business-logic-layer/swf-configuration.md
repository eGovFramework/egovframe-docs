# SWF Configuration

Spring Web Flow의 설정 항목(시스템 설정, MVC 연동, Securing Flows, Flow Managed Persistence)은 [The IoC Container](https://docs.spring.io/spring-framework/reference/6.2/core/beans.html)(Bean 정의), [Spring Framework 6.2 - Spring MVC](https://docs.spring.io/spring-framework/reference/6.2/web/webmvc.html) 및 [DispatcherServlet](https://docs.spring.io/spring-framework/reference/6.2/web/webmvc/dispatcher-servlet.html)과의 통합을 전제로 한다. 각 하위 문서에서 상세 설정을 설명한다.
